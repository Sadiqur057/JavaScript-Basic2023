Dom manipulation

The color can be changed using dom manipulation

DOM converts html page javascript object and can be accessed like java script object

DOM tree refers to the HTML page where all teh nodes are objects. There can be 3 main types of nodes in the DOM tree

  • text nodes
  • element nodes
  • comment nodes

In an HTML page, root is html, head and body are children

text is the leaf node

Walking the DOM

  • document.body => page body
  • document.getElement => page html tag
  • document.head => html head
  • document.title => return string
